# Verse of the Day
Bible Verse of the Day

This was implemented using JetBrains Rider on MacOS.

## Requirements
- GIT-SCM
- .NET Core
## Install & Run
```
$ git clone git@github.com:jameymcelveen/VerseOfTheDay.git
$ cd VerseOfDay
$ dotnet run
```
browse to [https://localhost:5001](https://localhost:5001)
